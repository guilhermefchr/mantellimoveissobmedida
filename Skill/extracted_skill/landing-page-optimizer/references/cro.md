# CRO / Conversão — Checklist e Referência

## Anatomia de uma Landing Page de Alta Conversão

```
┌─────────────────────────────────────┐
│  HERO (Above the Fold)              │
│  - Headline principal (proposta)    │
│  - Subheadline (benefício)         │
│  - CTA primário                     │
│  - Imagem/vídeo de suporte         │
├─────────────────────────────────────┤
│  PROVA SOCIAL                       │
│  - Logos de clientes / mídia       │
│  - Número de clientes / resultados │
├─────────────────────────────────────┤
│  PROBLEMA → SOLUÇÃO                 │
│  - Agitação do problema            │
│  - Apresentação da solução         │
├─────────────────────────────────────┤
│  BENEFÍCIOS / FEATURES              │
│  - Benefícios > Features           │
│  - 3–6 pontos principais           │
├─────────────────────────────────────┤
│  PROVA SOCIAL PROFUNDA              │
│  - Depoimentos com foto + nome     │
│  - Resultados específicos          │
│  - Cases ou estudos                │
├─────────────────────────────────────┤
│  CTA SECUNDÁRIO                     │
│  - Repetir oferta + urgência       │
├─────────────────────────────────────┤
│  FAQ / OBJEÇÕES                     │
│  - Responder 5–7 objeções comuns   │
├─────────────────────────────────────┤
│  CTA FINAL + GARANTIA              │
└─────────────────────────────────────┘
```

## Checklist CRO

### 🎯 Headline e Proposta de Valor
- [ ] Headline comunica o benefício principal em ≤ 10 palavras?
- [ ] Proposta de valor clara em 5 segundos (teste dos 5 segundos)?
- [ ] Headline fala sobre o CLIENTE, não sobre a empresa?
- [ ] Subheadline complementa e aprofunda o headline?
- [ ] Evita jargão e linguagem corporativa?

**Formulas de headline que convertem:**
- `[Resultado desejado] sem [maior objeção]` → "Emagreça 10kg sem academia"
- `Como [persona] conseguiu [resultado] em [tempo]` → "Como freelancers conseguiram dobrar faturamento em 60 dias"
- `[Número] [resultado] em [tempo]` → "3.000 leads em 30 dias"
- Pergunta que ressoa: `Cansado de [dor]?`

---

### 📣 CTA (Call to Action)
- [ ] Botão CTA visível sem rolar a página?
- [ ] Texto do botão é específico (não apenas "Clique aqui" ou "Enviar")?
- [ ] Cor do botão contrasta com o fundo (não apenas cor da marca)?
- [ ] CTA repetido pelo menos 3x ao longo da página?
- [ ] Há micro-copy abaixo do botão para reduzir friccão? ("Sem cartão de crédito", "Cancele quando quiser")
- [ ] Tamanho do botão adequado (especialmente mobile)?

**Exemplos de CTA de alta conversão:**
```
❌ "Enviar"
✅ "Quero meu acesso gratuito"

❌ "Saiba mais"
✅ "Ver demonstração agora"

❌ "Comprar"
✅ "Sim! Quero transformar meu negócio"
```

---

### 🏆 Prova Social
- [ ] Depoimentos com foto real, nome completo, e cargo/empresa?
- [ ] Resultados específicos nos depoimentos (não genéricos como "adorei!")?
- [ ] Logos de clientes conhecidos ou mídia?
- [ ] Número de clientes/usuários/reviews visível?
- [ ] Rating/estrelas se aplicável?
- [ ] Prova social posicionada ANTES do CTA principal?

---

### 😰 Tratamento de Objeções
- [ ] Seção de FAQ responde as 5 principais objeções?
- [ ] Garantia visível e clara?
- [ ] Política de privacidade linkada perto de formulários?
- [ ] Selos de segurança próximos ao checkout/formulário?
- [ ] Prova de resultado antes vs. depois (se aplicável)?

---

### 📝 Formulários
- [ ] Formulário pede apenas campos ESSENCIAIS?
- [ ] Cada campo tem label claro (não apenas placeholder)?
- [ ] Mensagens de erro são amigáveis e específicas?
- [ ] Botão de submit tem texto de ação (não "Submit")?
- [ ] Há indicação de privacidade próxima ao email?

**Regra de ouro**: Cada campo adicional reduz conversão ~10%. Peça apenas nome + email se possível.

---

### ⚡ Urgência e Escassez
- [ ] Há elemento de urgência real e crível? (não fake countdown)
- [ ] Limite de vagas ou bônus por tempo?
- [ ] Data específica de expiração da oferta?

---

### 🎨 Design e UX
- [ ] Hierarquia visual clara (o olho sabe onde ir)?
- [ ] Uma única ação principal por página?
- [ ] Sem menu de navegação que distraia? (LP deve ser sem menu)
- [ ] Contraste de texto adequado (WCAG AA: 4.5:1)?
- [ ] Espaço em branco suficiente para respirar?
- [ ] Acima da dobra não está "poluída" com informação demais?

---

### 📱 Mobile CRO
- [ ] CTA visível above the fold no mobile?
- [ ] Texto legível sem zoom no mobile?
- [ ] Formulários fáceis de preencher no teclado mobile?
- [ ] Imagens não cortadas ou distorcidas no mobile?
- [ ] Sticky CTA ou sticky header no mobile?

---

## Quick Wins de Maior Impacto

| Mudança | Impacto Esperado |
|---------|-----------------|
| Reescrever headline com benefício claro | +15–40% conversão |
| Melhorar texto do botão CTA | +10–25% CTR |
| Adicionar depoimento com resultado específico | +10–20% conversão |
| Remover campos desnecessários do formulário | +10–30% por campo removido |
| Adicionar garantia de devolução | +5–15% conversão |
| Adicionar número social ("12.000 alunos") | +5–15% confiança |
| Sticky CTA no mobile | +10–20% em mobile |
